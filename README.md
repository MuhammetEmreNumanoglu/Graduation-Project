## ⚙️ Kurulum

Öncelikle projeyi klonlayın:

masaüstüne boş bir klasör oluşturun
bu klasöre cmd den ulaşın
sonrasında şu kodu yazın
git clone https://github.com/kullaniciadi/proje-adi.git

bu projeyi bir editörden açın visual studio code vs.
sonra terminalde projenin yolunu bulun

cd proje-adi

🔹 Sanal Ortam Oluşturma
python -m venv venv

🔹 Sanal Ortamı Aktifleştirme
cd (projenin_yeri)\venv\Scripts\activate

🔹 Projeyi Çalıştırma
cd ..
cd ..
cd men
python manage.py runserver
